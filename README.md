## Quick Start

```bash
# Clone the repository
git clone https://github.com/techpotli/techpotli-dashboard.git

# Navigate to the project
cd techpotli-dashboard

# Install dependencies
npm install

# Start the development server
npm run dev

# Open in browser
http://localhost:3000
```
